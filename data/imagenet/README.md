## ImageNet Preprocessed Data

### Overview

The ImageNet data used in this project is derived from the **ImageNet validation set** and is preprocessed to support efficient evaluation with Adaptive AutoEval.

It consists of the following components:

- **Ground-truth correctness labels**  
  Binary indicators specifying whether each model prediction is correct.

- **Synthetic annotator predictions (AutoEval scores)**  
  Estimated performance signals generated by the AutoEval framework.

- **Outputs from 5 ResNet models**  
  Predictions from multiple pretrained ResNet architectures used for evaluation.

All data is stored as **NumPy arrays (`.npy`)** to enable fast loading and efficient computation.
